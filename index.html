Automatic Syllabification

An automatic syllabifier is an algorithm that takes words and produces a linguistically correct breakdown of the word without being provided syllable information beforehand. Syllables are sequences of phonemes that "go together" in the sense that they contain a vowel center followed and/or preceded by short consonant sounds. Natural languages have their own unique syllable patterns, and these patterns are related to the set of sounds available in the language. When working with language data, especially in audio, it can be very useful to break utterances down by syllable, because syllables represent a unit of sound that is below the word level but above the phoneme level, which is useful for identifying properties of sound like stress.

Syllabifying IPA transcribed text is fairly easy. Look at this link for example, an automatic syllabification algorithm by Penn University based on text representation of words. Despite being an important aspect of audio analysis, the availability of an application that can automatically syllabify speech is limited. There are some algorithms that apply spectral analysis on the input audio file to identify syllable boundaries, but these require advanced mathematics to use and are not very intuitive. The hardest part usually is to obtain an alignment for the audio file, then a syllabification can be obtained by applying the maximum onset principle . The following application aims to gather these ideas and provide a convenient program that can automatically syllabify text and audio. Currently, it can produce automatic syllabifications for English text, but I would like to add support for German text and Arabic audio in the future!
